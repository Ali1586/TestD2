## Mynewapp 
- Smidig och lätt system för säkerställa att testdata är nogranat i integrationen med GDPR

## Docker 
- Appen är i isolerade Docker container med hjälp av Python 3 (VS)



### Förutsättningar
- Docker och Docker Compose
- Python 3.9+
- Git
